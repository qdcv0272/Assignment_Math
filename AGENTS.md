## 프로젝트 개요

- React + TypeScript + Vite + PandaCSS CSS 프론트엔드 프로젝트

## 코딩 컨벤션

- PandaCSS 는 일단 사용 금지 사용자가 잘 모르는 영역임
- 불필요한 `useEffect` 사용을 지양하고, 상태 관리 라이브러리(Zustand)를 사용하세요.

## 명령어

- 패키지 설치: `npm install`
- 개발 서버 실행: `npm run dev`
- 빌드: `npm run build`

## 작업 규칙

- 새로운 컴포넌트를 만들 때는 `components/` 디렉토리에 생성하세요.
- prettier 규칙에 맞게
