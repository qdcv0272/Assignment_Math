## 학습 완료

- `달빛약속`은 한글 키워드 기반 언어입니다.
- `약속` = 함수/메서드, `보여주기` = 출력, `클래스`/`새`/`자신`/`상위` = 객체지향.
- 숫자 연산은 `+ - * /`, `**` 제곱, `%` 나머지, `//` 정수 나눗셈을 지원합니다.
- 변수는 한글 이름 가능, `변수 = 값` 형태로 선언·할당합니다.
- `약속, 이름 (매개변수)`로 정의하고 `반환하기`로 결과 반환.
- 클래스 생성자는 `약속, __준비__`, 상속은 `클래스, 자식(부모)` 형태.
- 한국어 조사 변형도 `약속` 이름에 `/`를 쓰는 방식으로 처리할 수 있습니다.
